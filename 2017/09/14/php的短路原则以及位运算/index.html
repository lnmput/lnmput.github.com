发表于 2017-09-14 更新于 2021-09-27

短路作用

当 || 前面的值为 false 时， || 后的表达式才会被执行， 此时结果为5；

当 || 前面的值为 true 时， || 后的表达式不会被执行， 此时结果就找不到 $b

当 && 前面的值为false 时， 后面的表达式不会被执行

只有当 && 前面的值为true 时， 后面的表达式才会被执行

位运算

左移 <<

左移运算的实质是将对应的数据的二进制值逐位左移若干位，并在空出的位置上填0，最高位溢出并舍弃。
左移一位, 就实现了乘2运算

右移 >>

可以实现 除 2 运算
